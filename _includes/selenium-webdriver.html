Selenium WebDriver

Automatizando testes com Selenium WebDriver em C# utilizando Page Objects.

O que é Selenium ?

Selenium não é uma ferramenta e nem uma biblioteca ele é um facilitador para automação web, ou seja, ele automatiza aplicações web para fins de teste, mas com certeza não se limita a apenas isso.

O que é Page Objects ?

A proposta desse padrão é de criar um objeto para cada pagina web do projeto que seja testado, e utilizando orientação a objetos, encapsulamos em cada classe os atributos e métodos, como campos e ações de cada página. Por exemplo, para cada página a ser testada em nossos testes teremos uma classe que mapeia cada uma das ações e campos.
